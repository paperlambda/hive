# Hexagons
Basic addition and mapping for hexagonal grids

## Prerequisites
- Node.js
- MongoDB

## Up & Running
```
npm install
npm run build 
npm start
```

## Limitation
1. Have NOT yet support negative x and/or y axis
2. No delete function (Case 3)
